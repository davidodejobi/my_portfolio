## Run Locally

Clone the project
```bash
  git clone https://github.com/davidaodejobi/bio
```

Go to the project directory

```bash
  cd my_portfolio
```

Install dependencies
```bash
  flutter pub get
```

Run project
```bash
  cd run
```
